# lua

Introductory programs in Lua

# EXAMPLE

```
$ lua hello.lua
Hello World!
```

# REQUIREMENTS

* [lua](http://www.lua.org/) 5+

## Optional

* [checkmake](https://github.com/mrtazz/checkmake)
* [GNU findutils](https://www.gnu.org/software/findutils/)
* [GNU make](https://www.gnu.org/software/make/)
