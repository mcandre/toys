# gbash: Example GNU bash scripts

# Hello World

```console
$ ./lib/hello
```

# REQUIREMENTS

# DEVELOPMENT

## Build: Lint and Test

```console
$ vast [v-build]
```

## Test

```console
$ vast v-test
```

## Lint

```console
$ vast v-lint
```
