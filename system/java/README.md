# jvm: JVM examples

# EXAMPLE

```console
$ cd java/hello
$ gradle --warning-mode all build
$ bin/hello
Hello World!
```

# REQUIREMENTS

* [OpenJDK](https://openjdk.org/) 18.0.2.1+
* [Gradle](https://gradle.org/) 7.5.1+
* [Node.js](https://nodejs.org/en) 16.14.2+ with `npm install -g snyk1.996.0`
